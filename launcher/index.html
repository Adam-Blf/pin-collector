Pin Collector • Launcher

Petit panneau qui t’aide à lancer le projet en 2 commandes

1. Installer les dépendances

Ouvre un terminal dans le dossier du projet pin-collector puis colle la commande ci dessous.

pip install -r requirements.txt

2. Lancer l’app

Une fois les dépendances installées, lance l’interface.

streamlit run streamlit_app.py

Scripts prêts à l’emploi

Tu peux aussi utiliser un script si tu préfères.

  • start_windows.bat pour Windows
  • setup_and_run.ps1 pour PowerShell
  • start_unix.sh pour macOS et Linux

Les scripts supposent que tu es dans le dossier pin-collector.

Conseils rapides

  • Si pip ne marche pas, essaie python -m pip.
  • Si streamlit n’est pas reconnu, ferme et rouvre le terminal.
  • Tu peux créer un venv si tu veux garder l’environnement propre.
